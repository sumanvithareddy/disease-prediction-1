# 🏥 MediPredict AI - Disease Prediction System

A modern Disease Prediction System built using **HTML, CSS, and JavaScript**.

This educational web application predicts possible diseases based on user-selected symptoms and provides health information including disease description, causes, treatment suggestions, prevention tips, and recommended diet.

# 📊 Prediction Process

1. User selects symptoms.

2. JavaScript compares selected symptoms with disease database.

3. Best matching disease is found.

4. Confidence score is calculated.

5. Disease information is displayed.
